# NinjaGame-frontend
School project front-end in vue.js

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
### Run cypress e2e tests
```
npx cypress open
```
